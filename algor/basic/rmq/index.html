    RMQ¶

    区间最值问题。

    可以考虑线段树解决。

    不过有一个更好写一点的方法，\(\text{O}(n \log n)\) 预处理，\(\text{O}(1)\) 询问，但是不能带修。

    这就是 ST 表。

    利用倍增的思想考虑：

    设 \(f_{i,j}\) 表示从 \(i\) 开始的 \(2^j\) 个数当中的最值。

    类似线段树，把两个半区间的信息上传到大区间。

    那么 \(f_{i,j}=\max(f_{i,j-1},f_{i+2^{j-1},j-1})\)

    询问区间 \([l,r]\) 的最值的时候，找到一个 \(k\) 使得 \(2^k \le r-l+1 \le 2^{k+1}\)。

    那么 \(ans=\max(f_{l,k},f_{r-2^k+1,k})\)。这两个区间会覆盖，但是没有影响。

    const int si=1e5+10;
int st[si][20];
int a[si],Log[si];
inline void ST_prework(){
    for(register int i=2;i<=n;++i){
        Log[i]=Log[i>>1]+1;
    }
    for(register int i=1;i<=n;++i){
        st[i][0]=a[i];
    }
    for(register int j=1;j<=Log[n];++j){
        for(register int i=1;i<=(n+1)-(1<<j);++i){ // 注意循序
            st[i][j]=max(st[i][j-1],st[i+(1<<(j-1))][j-1]);
        }
    }
}
inline int query(int l,int r){
    int k=Log[r-l+1];
    return max(f[l][k],f[r-(1<<k)+1][k]);
}
