    Astyle 配置

    怎么用感觉不用说，自己看下 docs 就行了。

    不过关于是否打大括号，这个就纯看我心情，不过这个 format 之后和我正常码风基本一致了？

    Astyle-options.txt

    还是简单说一下代码风格比较好。

    括号是 Google 式的大括号不换行，for if while 之后不会有空格。

    分号，逗号，运算符后会有空格，for if while 后会空行，或者是按功能分块。

    不使用万能头，头文件按长度，字典序为关键字排序。

    #define endl '\n'，通常关掉流同步，有时候会不打括号直接写一行。

    不过分依赖于 STL。
